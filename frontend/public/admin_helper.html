Kullanım Talimatları
Admin Kullanıcısı Oluşturmak İçin:
  1. Soldaki formu doldurun (Ad Soyad, E-posta, Şifre)
  2. Güvenlik anahtarı: Kadire5866ModeRn2024
  3. "Admin Kullanıcısı Oluştur" butonuna tıklayın
Mevcut Kullanıcıyı Admin Yapmak İçin:
  1. Önce siteye normal kayıt olun
  2. Sonra bu sayfaya geri dönün
  3. E-posta ve şifrenizle giriş yapın
  4. "Giriş Yap ve Admin Yap" butonuna tıklayın
Not: Admin olduktan sonra sayfayı yenileyip siteye tekrar giriş yapmalısınız.
